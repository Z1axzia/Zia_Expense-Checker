# Zia_Expense-Checker
# 💰 Expense Management System

## Overview

A Python-based Expense Management System that allows users to record, track, and manage personal expenses. The application stores expense data in CSV format and provides an easy way to monitor spending habits through a simple command-line interface.

## Features

- Add new expenses
- View all recorded expenses
- Search expenses by category
- Calculate total spending
- Store expense records in CSV files
- Simple and user-friendly menu system

## Technologies

- Python 3
- CSV Module
- File Handling
- Object-Oriented Programming (OOP)
